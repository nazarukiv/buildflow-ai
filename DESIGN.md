## 1. Visual Direction

Style:
Dark, high-contrast interface with strong accent colors

Mood:
Modern, technical, slightly aggressive (developer-focused, not corporate)

Principle:
Minimal elements, maximum clarity

---

## 2. Color System

Primary background:
- #131313 (main canvas)

Primary text:
- #ffffff (main content)

Secondary text:
- #949494 (metadata, labels)

Accent colors:
- #3cffd0 (primary highlight)
- #5200ff (secondary highlight)

Secondary surfaces:
- #2d2d2d (cards, panels)

Rules:
- No gradients
- No soft shadows
- Use solid colors only

---

## 3. Typography

Display:
- Heavy, bold font for large titles

Body:
- Clean sans-serif for readability

Labels:
- Uppercase, spaced letters

Guidelines:
- Titles = strong and large
- Body = simple and readable
- Labels = compact and structured

---

## 4. Components

Buttons:
- Rounded (pill style)
- Primary = accent color background
- Secondary = dark background

Cards:
- Rounded corners (20px+)
- Thin border (1px)
- No shadow

Labels:
- Small, uppercase
- Used for tags and metadata

Tables:
- Clean rows
- No heavy borders
- Focus on readability

---

## 5. Layout

Spacing system:
- Base unit: 8px

Structure:
- Vertical stacking
- Clear separation between sections

Rules:
- Avoid clutter
- Keep consistent spacing
- Use empty space for clarity

---

## 6. Interaction

Hover:
- Change text or border color only

Active:
- Slight color change

Focus:
- Visible but minimal

Animation:
- Fast and subtle (150–200ms)

---

## 7. Design Principles

- Simplicity over decoration
- Consistency across components
- Clarity over creativity
- Functional first, visual second

---

## 8. Do / Don’t

Do:
- Use dark background
- Use accent colors for important elements
- Keep UI clean and structured

Don’t:
- Use gradients
- Use heavy shadows
- Mix too many colors
- Overcomplicate layouts

---

## 9. Responsiveness

Mobile:
- Single column
- Full-width components

Tablet:
- 2-column layout where needed

Desktop:
- Structured layout with clear sections

---

## 10. Notes

This design system is built for:
- developer tools
- dashboards
- AI-driven applications

It prioritizes:
- speed
- clarity
- usability
